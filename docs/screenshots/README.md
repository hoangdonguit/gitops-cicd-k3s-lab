# Screenshots

This directory stores selected verification screenshots for the `gitops-cicd-k3s-lab` portfolio project.

Recommended screenshots:

- `github-actions-ci-success.png`: GitHub Actions `ci-gitops` workflow success.
- `argocd-app-synced-healthy.png`: ArgoCD Application `portfolio-gitops-cicd-demo` in `Synced / Healthy` state.
- `k3s-runtime-verification.png`: Kubernetes runtime verification with pod, NetworkPolicy, `/version`, and `/healthz`.
- `security-context-verification.png`: Kubernetes pod and container securityContext verification.

Screenshots should avoid exposing secrets, tokens, credentials, or unrelated private project details.
